Recording-ready training package

Lumo LMS Admin Video Tutorial Pack

This is the practical production pack for recording admin training on the Lumo LMS. It covers the actual surfaces that exist now: dashboard flow, Content Library, English Studio, Lesson Studio, and mallam/learner operations.

What’s included
  • Series outline and recommended viewing order
  • Per-video scripts and narration copy
  • Chapter timestamps and run-of-show structure
  • Shot lists for screen recording
  • Companion production checklist
Use this when
  • You need to onboard LMS admins fast
  • You need training assets before final videos are recorded
  • You want consistent recordings that match the current product

Production notes

How to turn this into real video fast

  • Record in chapters, not one giant take
  • Use the per-video shot lists instead of improvising
  • Export this file and the dashboard guide as PDFs for non-video sharing
  • Re-record chapters if the UI changed; don’t fake it with vague narration
