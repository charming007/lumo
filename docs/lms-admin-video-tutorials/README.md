# Lumo LMS Admin Video Tutorial Pack

_Last updated: 2026-04-14_

This is the **recording-ready tutorial pack** for Lumo LMS admins and operators.

## What this pack is for

Use this pack when you need to turn the current LMS into a practical training series for:

- LMS admins
- content ops teams
- curriculum leads
- implementation managers
- mallam supervisors
- onboarding staff supporting learners on the ground

It is designed for the annoying real-world constraint we have right now:
**we may not be recording the final polished videos yet.**
So this pack ships the next best thing:

- a clear video series plan
- per-video run-of-show
- chapter timestamps
- shot lists and screen cues
- narration copy
- presenter notes
- companion reading pages
- production checklists so anyone can record later without winging it

## Recording guidance

### Recording style

Keep the videos:

- practical, not salesy
- screen-first
- click-by-click
- grounded in the current UI
- explicit about what each page is for
- strict about release/readiness logic

### Voice/tone

Use a calm, capable operator voice.
Do not narrate like a marketing reel.
This is training, not theatre.

### Best production pattern

For each video:

1. Record a clean intro take.
2. Record UI actions in short segments.
3. Use the chapter list below as edit points.
4. Cut mistakes aggressively.
5. Add captions using the narration copy as the transcript base.
