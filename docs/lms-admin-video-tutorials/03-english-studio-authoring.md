# Video 3 — English Studio and fast lesson authoring

**Audience:** English content editors, curriculum authors, LMS admins  
**Target runtime:** 10-12 minutes  
**Goal:** show how English Studio helps authors create better English lessons with stronger structure and readiness controls.

## Outcome

By the end of this video, the viewer should be able to:

- explain what English Studio does differently from Content Library
- create an English lesson from a blueprint/preset
- understand readiness checks and blockers
- know when to continue in Lesson Studio

## Run of show

### Segment 1 — Why English Studio exists
**Narration:**
> English Studio exists because English lesson quality needs more structure than a simple title-and-duration form. It adds objectives, vocabulary, activity sequencing, localization support, assessment context, and release checks.

### Segment 2 — English Studio vs Content Library
**Narration:**
> Content Library owns the official structure. English Studio sits on top of that structure for English-specific authoring. It does not replace the curriculum spine. It strengthens the lesson before it gets written back into the live content lane.

### Segment 3 — Starting from a preset
**Narration:**
> Start with a quick preset or generated blueprint when it helps. That is faster and usually smarter than starting from a blank box. You can always refine the flow afterward.

### Segment 4 — Core lesson settings
**Narration:**
> Choose the correct English module first. Then set the title, duration, delivery mode, and publish state. If the module is weak or the title is vague, the readiness logic will rightly push back.

### Segment 5 — Objective, vocabulary, and localization
**Narration:**
> Review the generated objective and vocabulary focus, then tune localization. For this product, support language and localization notes are not optional fluff. They matter for real-world delivery.

### Segment 6 — Assessment link and readiness
**Narration:**
> English Studio surfaces the linked assessment context and the readiness score. Pay attention to blockers. If the module has no assessment gate or the lesson is undercooked, do not mark it approved just because you’re in a hurry.

### Segment 7 — Editing the activity spine
**Narration:**
> The activity spine is where the lesson becomes teachable. Add, reorder, duplicate, or remove steps so the flow actually works in a real classroom or pod setting.

### Segment 8 — When to open full Lesson Studio
**Narration:**
> English Studio is the better first pass for structured English lesson authoring. But if you need full interactivity, answer choices, media cues, detailed evidence, or richer assessment setup, jump into Lesson Studio.
