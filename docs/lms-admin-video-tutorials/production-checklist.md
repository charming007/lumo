# Lumo LMS admin tutorial production checklist

Use this when turning the script pack into actual recorded videos.

## Before recording

- [ ] Confirm the LMS build matches the docs in this pack
- [ ] Confirm route names still match sidebar labels
- [ ] Prepare a clean demo dataset
- [ ] Remove junk/demo records you do not want on camera
- [ ] Choose one browser window size and stick to it
- [ ] Turn off notifications
- [ ] Prepare zoom level for readable recording
- [ ] Decide whether cursor highlighting will be used
- [ ] Prepare intro/outro title cards

## During recording

- [ ] Record short sections per chapter, not one giant take
- [ ] Keep cursor movement deliberate
- [ ] Pause before important clicks
- [ ] Read the narration, but do not sound like a hostage
- [ ] Show the relevant panel long enough for editors to cut cleanly
- [ ] Re-record any segment where the UI state becomes confusing

## Editing

- [ ] Add chapter cards using the timestamps in each script
- [ ] Use the narration as the caption base
- [ ] Trim dead air and cursor wandering
- [ ] Add zoom-ins only where the UI is dense
- [ ] Export landscape master files
- [ ] Export lighter share copies if needed for WhatsApp/Telegram distribution

## Publish package

- [ ] Video files exported
- [ ] PDFs exported from `LMS_DASHBOARD_GUIDE.html` and `LMS_ADMIN_VIDEO_TUTORIAL_PACK.html`
- [ ] Markdown source retained for updates
- [ ] Final distribution location agreed

## If the UI changed after recording

Do not patch it with a vague voiceover if the clicks are now wrong.
Re-record the affected chapter. Training videos get stale fast when people can visibly see the mismatch.
