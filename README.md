# Lumo

Voice-powered AI learning platform for children in low-resource environments.

## Products

- `apps/learner-tablet` — Flutter learner app for shared Android tablets
- `apps/lms-web` — Next.js LMS/admin portal
- `services/api` — backend API and sync services
- `docs/` — product, architecture, roadmap, schema, and delivery docs

## V1 Goal

Pilot-ready MVP for Almajiri children in northern Nigeria (ages 8–12), focused on:

- Foundational English oral literacy
- Basic numeracy
- Hygiene, safety, and life skills

## Product principles

- Voice-first
- Offline-first lesson playback with delayed sync
- Guided lessons before open conversation
- Shared-tablet profile switching
- Measurable learning outcomes
- Human-reviewed content improvement
